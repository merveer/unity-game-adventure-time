



      The Pixel Man

Demo game for Moenen-Pixel-Collection. Please enjoy : )


The Pixel Man is a platformer demo game for Moenen Pixel Collection. 
Kill the boss at the bottom stage to get a super coin will beat the game. 
Collect 4 coins to win perfectly.


Please feel free to modify and use the source code and assets in that game, but not sell the game without change it.
